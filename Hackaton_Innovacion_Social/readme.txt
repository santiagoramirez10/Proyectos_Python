En este proyecto se programan una serie de clases para cumplir con un reto de tema social. Se tienen datos de estudiantes generados aleatoriamente y no tienen que ver con la realidad. Entre los datos generados aleatoriamente se tienen las direcciones de residencia, su localidad, zona, riesgos sociales de las zonas entre otros datos que son mostrados en el proyecto. Además de ello se crea un archivo de Excel con todos los datos y también una base de datos por medio de librerías.
